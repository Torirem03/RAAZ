Features
  • Spring Boot
  • Lombok
  • Thymeleaf
  • JPA
  • MySQL

Our Tea Inventory System, built with Spring Boot, JPA, Lombok, MySQL, and Bootstrap, is an easy-to-use tool for tea businesses. It helps track your tea inventory, keeps data organized, and provides a simple interface for a smooth experience.

In our project, we harnessed the combined strength of GitHub and Discord for optimal collaboration. GitHub served as our version control hub, enabling simultaneous development through branching and pull requests. Discord facilitated real-time communication with dedicated channels for discussions and screen sharing.
